## Calculator Using Functions

A Python calculator program that performs basic arithmetic operations using functions.
It allows the user to choose an operation and handles division by zero safely.

# Concepts Used

- Functions

- Return Statements

- Conditional Statements

- User Input Handling

- Basic Arithmetic Operations

# Features

- Performs addition, subtraction, multiplication, and division

- Uses separate functions for each operation

- Handles division by zero gracefully

- Easy to read and modular code structure

# How to Run

- Save the file as calculator_functions.py
