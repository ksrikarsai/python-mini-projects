# Guessing Game

A simple Python number guessing game where the player has a limited number of tries to guess a secret number between 1 and 100.
It provides hints after each guess to guide the player.

## Concepts Used

- Random Number Generation

- For Loop

- Conditional Statements

- User Input Handling

- Break and Loop Control Flow

## Features

- Randomly generates a secret number

- Provides hints: too high or too low

- Limits the player to a fixed number of attempts

- Displays success or failure message

## How to Run

- Save the file as guessing_game.py

- Run the program using:

- python guessing_game.py
