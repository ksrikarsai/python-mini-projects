## Unit Converter

A simple Python program that converts values between common units using functions.
It supports distance, temperature, and weight conversions through a menu-driven interface.

# Concepts Used

- Functions

- Return Statements

- Conditional Statements

- User Input Handling

- Arithmetic Operations

# Features

- Converts kilometers to miles

- Converts Celsius to Fahrenheit

- Converts kilograms to pounds

- Menu-driven and easy to use

# How to Run

- Save the file as unit_converter.py

- Run the program using:

- python unit_converter.py
