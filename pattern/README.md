## Number Pattern Generator

A simple Python program that prints a number pattern where each row displays numbers from 1 up to the current row number.
It uses nested loops to generate a clean, incremental numeric pattern.

# Concepts Used

- For Loops

- Nested Loops

- Range Function

- User Input Handling

## Features

- Generates a structured number pattern

- Uses nested loops for pattern formation

- Clean and beginner-friendly logic

- Interactive input-based output

## How to Run

- Save the file as pattern.py

- Run the program using:

- python pattern.py
