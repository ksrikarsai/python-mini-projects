# Simple Login System

A basic Python login validation program that checks a username and password with a limited number of attempts.
It provides clear feedback on incorrect inputs and locks the account after multiple failed tries.

## Concepts Used

- For Loops

- Conditional Statements

- User Input Handling

- String Comparison

## Features

- Validates username and password

- Allows only a limited number of login attempts

- Provides specific error messages

- Locks the account after all attempts are used

## How to Run

- Save the file as login_system.py

- Run the program using:

- python login_system.py
