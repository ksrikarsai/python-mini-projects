# Basic Billing System

A simple Python program that generates a basic bill by taking item details as input.
It calculates the total amount, applies GST, and displays the final payable amount.

## Concepts Used

- User Input Handling

- Variables and Data Types

- Basic Arithmetic Operations

- Simple Console Output

## Features

- Calculates total cost based on quantity and price

- Automatically applies GST

- Displays a clear, formatted bill

- Beginner-friendly and easy to extend

## How to Run

- Save the file as billing_system.py

- Run the program using:

- python billing_system.py
